PHONICS FLUENCY PORTAL v1.2 - DEPLOYMENT NOTES

BEST BETA DEPLOYMENT:
Use the same method as Monster Math Arcade:
- Keep the ZIP package in Google Drive as the source file vault.
- Host the unzipped PWA folder on HTTPS static hosting for the best Lenovo tablet install experience.
- Use the single-file fallback only when hosting is not ready.

RECOMMENDED GOOGLE DRIVE FOLDER:
Kids Learning Apps/
  Phonics Fluency Portal/
    Versions/
    Backups/
    Beta Notes/

UPDATE FLOW:
1. Open Parent Dashboard.
2. Click Download Parent Backup.
3. Save the JSON file to Google Drive > Kids Learning Apps > Phonics Fluency Portal > Backups.
4. Deploy/open the new app version.
5. If progress is missing, open Parent Dashboard and Import Backup.

LENOVO INSTALL FLOW ONCE HOSTED:
1. Open the hosted HTTPS URL in Chrome.
2. Tap Chrome three-dot menu.
3. Tap Install app or Add to Home screen.
4. Name it Fluency Portal.
5. Launch from the home screen.

GOOGLE DRIVE ROLE:
Google Drive is ideal for backups, version storage, and family file transfer.
Google Drive is not ideal as true PWA hosting because install/offline features need manifest.webmanifest and sw.js served correctly from the same HTTPS origin.

SHARED PARENT DASHBOARD LATER:
This app now exports a structured JSON backup with:
- appId: phonics_fluency_portal
- sharedParentDashboardReady: true
Monster Math Arcade can use a similar export format. Later, both can feed one dashboard.
